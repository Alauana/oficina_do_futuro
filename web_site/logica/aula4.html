Repetição (Loops)

Para que repetir?

Computadores são ótimos em fazer tarefas chatas repetidamente sem cansar.

Imagine escrever "Eu não vou jogar bolinha de papel" 100 vezes no quadro.

O Laço ENQUANTO (While)

Repete enquanto uma condição for verdadeira.


        contador = 0

        enquanto (contador < 5) {
            escreva("Repetindo...");
            contador = contador + 1; // Importante!
        }

O Laço PARA (For)

Quando você sabe exatamente quantas vezes quer repetir.


        // Para i de 1 até 10 faça:
        para (i = 1; i <= 10; i++) {
            escreva("Número: " + i);
        }
