Novos Brinquedos

  • LDR (Fotoresistor): Sensor de Luz.
  • Micro Servo: Motor que gira para uma posição exata (0° a 180°).

Controlando o Servo

Precisamos de uma biblioteca (uma caixa de ferramentas pronta).


            #include 

            Servo meuServo;

            void setup() {
            meuServo.attach(9); // Servo no pino 9
            }

            void loop() {
            meuServo.write(90); // Vá para 90 graus
            }

Projeto: A Catraca Automática

Se o sensor de presença (PIR) ou botão for ativado:

  1. Acenda o LED Verde.
  2. Gire o Servo para 90° (Abrir).
  3. Espere 3 segundos.
  4. Gire o Servo para 0° (Fechar) e apague LED.
