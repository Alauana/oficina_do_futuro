PWM & RGB

Aula 04: Pintando com Luz

O que é PWM?

(Pulse Width Modulation)

É uma técnica para simular voltagem analógica.

Basicamente: Ligar e desligar MUITO rápido para parecer mais fraco.

Comando Mágico

Funciona apenas nos pinos com o tio (~): 3, 5, 6, 9, 10, 11.


            // Valor vai de 0 (apagado) a 255 (máximo)
            analogWrite(9, 127); // 50% de brilho

O LED RGB

São 3 LEDs em um só: Red, Green, Blue.

Misturando os valores (0-255) de cada perna, criamos qualquer cor!

Atividade: Mesa de DJ de Luz

Use 3 potenciômetros para controlar as 3 cores do LED RGB.

Dica: Use a função map() para transformar 0-1023 (pot) em 0-255 (led).
