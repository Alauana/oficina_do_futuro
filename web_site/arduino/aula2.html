Alauana - Instrutora

Digital Read (Ler Digital)

O Arduino pergunta: "Tem energia chegando aqui?"

  • HIGH (1): Sim, botão apertado (5V).
  • LOW (0): Não, botão solto (0V).

A Lógica IF (Se)


            int estadoBotao = digitalRead(2);

            if (estadoBotao == HIGH) {
                // Se apertou...
                digitalWrite(13, HIGH); // Liga LED
            } else {
                // Senão...
                digitalWrite(13, LOW);  // Desliga LED
            }

Atividade Prática

Crie um interruptor de luz.

Componentes: 1 Botão (Pushbutton), 1 Resistor de 10k (Pull-down), 1 LED.

Dica: Cuidado com o resistor pull-down, sem ele o Arduino fica "confuso"!
